# Seeding Guidelines — Telegram

Seeding = natural context around the signal. **Not** hype. **Not** a second signal.

## Purpose

- Explain *why* the setup matters in current session
- Educate on risk and management
- Build trust — not urgency

## Structure (1–3 short messages max)

### Message A — Context (optional, before signal)

```text
London session open — XAUUSD watching {level} for {setup_type}.
Volatility {normal|elevated}. {news_note if any}
```

### Message B — After signal (optional)

```text
Signal posted. Suggested management: consider partial at 1R if momentum slows.
Always size for your account. Questions → DM.
```

## Tone Rules

| Do | Don't |
|----|-------|
| Factual session notes | Guaranteed outcomes |
| Risk reminders | Urgency / FOMO |
| Historical context with disclaimer | "Easy money" framing |
| Invite DM for questions | Pressure to increase size |
| One setup focus | Multiple conflicting calls |

## news_note Examples

- `No high-impact USD news in next 30 min.`
- `High-impact USD news in 45 min — elevated caution.`
- `NFP week — wider spreads possible.`

## Lint Check (before publish)

Reject seeding copy containing:
- guarantee, sure win, 100%
- don't miss, last chance, all in
- passive income, get rich

## Lot Reference

Reference client group tiers without dictating individual account sizes:

```text
Conservative tier: see pinned lot guide. Standard tier: see pinned lot guide.
```

Do not publish actual client account balances or credentials.
